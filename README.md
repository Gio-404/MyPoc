## 📖 介绍

本仓库收集并整理了互联网公开的漏洞检测POC（Proof of Concept），采用"古法编程"将其转化为可直接运行的Python脚本。所有POC脚本仅供检测不含利用。

## ✨ 特点

- 🔧 将公开POC转化为标准化的Python脚本
- 📝 统一的脚本格式和参数规范
- 🚀 支持批量检测
- 🔒 仅用于授权测试和安全研究

## 📋 环境要求

- Python 3.7+
- 依赖库：requests, urllib3等
